## Overview

**ReportGen 2.0** is a web-based Student Report Card Management System developed using core Web Programming technologies. The project demonstrates how frontend, backend, and database components can be integrated to build a functional academic record management system.

The system allows users to add, view, and delete student academic records through a structured and user-friendly web interface.

---

## ReportGen vs ReportGen 2.0

### ReportGen (Core Java Version)

ReportGen was originally developed as a console-based Core Java application as part of the academic syllabus. Later, it was enhanced with a Java Swing GUI to simulate a real desktop application.

**Key Characteristics:**

* Built using Core Java
* Initially CMD-based, later upgraded to Java Swing GUI
* Uses file handling for data storage
* Works offline without database or internet
* Focused on Object-Oriented Programming fundamentals

---

### ReportGen 2.0 (Web Programming Version)

ReportGen 2.0 is the next evolution of the project, redesigned as a web-based academic management system using modern web technologies.

**Key Characteristics:**

* Built using HTML, CSS, JavaScript, PHP, and MySQL
* Multi-page web application
* Uses MySQL database for permanent data storage
* Dynamic student record management (Insert, View, Delete)
* Form validation and interactive UI features
* Designed with professional UI/UX principles

---

## Features

### Frontend

* Multiple web pages
* Structured student data entry form
* Dynamic student record table
* External CSS for layout, colors, hover effects, and box model
* Professional footer with academic and project details

### JavaScript Functionality

* Required field validation
* Email and numeric validation
* Alert and confirm popup messages
* Automatic total marks calculation

### Backend (PHP + MySQL)

* Processes form submissions
* Performs database operations
* MySQL table with primary key for unique student identification
* Supports Insert, View, and Delete operations

## Objective

The main goal of ReportGen 2.0 is to demonstrate the practical implementation of full-stack web development concepts and show how academic management systems can be built using core web technologies.
This project also highlights the progression from a basic Java console application (ReportGen) to a database-driven web application (ReportGen 2.0).
